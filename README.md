# Happy Birthday Mini Website

An interactive birthday-greeting mini website built with **HTML, CSS, and JavaScript**.  
Designed as a simple, cute surprise page with animations and a typewriter-style message.

## Live Demo
- https://vashish09.github.io/HappyBirthdaySharu/

## Features
- Typewriter-style birthday message
- Responsive design (mobile + desktop)
- Easy to customize text and assets

## Tech Stack
- HTML5
- CSS3 (animations)
- JavaScript

## Run Locally

### Clone
```bash
git clone https://github.com/vashish09/HappyBirthdaySharu.git
cd HappyBirthdaySharu
```

### Open in browser
You can open `index.html` directly, or run a local server (recommended):

```bash
python -m http.server 5500
```

Then visit:
- http://localhost:5500

## Customize
Common customization points:
- **Text / names / messages**: edit the HTML files and any message strings in JS
- **Colors / animations**: edit the CSS files
- **Images / GIFs**: replace assets and update file paths if needed
